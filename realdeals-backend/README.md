# RealDeals PFE

RealDeals is a full-stack e-commerce platform for a streetwear brand.  
It includes a customer website and an admin dashboard for managing products, categories, orders, clients, sales, and product images.

## Technologies

- React
- Vite
- Tailwind CSS
- Spring Boot
- Spring Security
- JWT
- MySQL
- JPA / Hibernate

## Features

- Product listing
- Product detail page
- Cart and checkout
- User authentication
- Admin dashboard
- Product image upload
- Multiple product images
- Product/category/order/client management
