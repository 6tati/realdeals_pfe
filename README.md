# RealDeals PFE

RealDeals is a full-stack e-commerce platform developed as a final year project.

The project is designed for a streetwear brand and contains two main parts:

- A public e-commerce website for clients
- An admin dashboard for managing products, categories, clients, and orders

## Project Objective

The objective of this project is to create a complete web application that helps a clothing/streetwear brand manage its online store in a structured way.

The platform allows users to browse products, view product details, use a shopping cart, and allows the administrator to manage the main data of the store.

## Technologies Used

### Frontend

- React.js
- Vite
- JavaScript
- CSS

### Backend

- Spring Boot
- Java
- Spring Security
- JWT Authentication
- REST API

### Database

- MySQL
- PowerAMC for database modeling
